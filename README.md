# 🌿 Raman Greens E-commerce Application

## 📌 Overview

Raman Greens is a modern e-commerce web application focused on delivering a seamless and responsive shopping experience. The project is being developed using the MERN stack architecture, with the frontend currently implemented and backend integration in progress.

---

## 🚀 Features

* 🛒 Browse products with a clean UI
* 🔍 View detailed product information
* ⚡ Fast performance using Vite
* 📱 Fully responsive design
* 🧩 Modular and scalable architecture

---

## 💻 Tech Stack

### Frontend

* React.js
* TypeScript
* Vite
* Tailwind CSS

### Backend *(In Progress)*

* Node.js
* Express.js

### Database *(Planned)*

* MongoDB

---

## 📂 Project Structure

```bash
src/
 ├── components/     # Reusable UI components
 ├── pages/          # Application pages
 ├── services/       # API calls (future backend integration)
 ├── hooks/          # Custom React hooks
 ├── utils/          # Helper functions
 ├── types/          # TypeScript interfaces
 ├── assets/         # Images and static files
```

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/DurgaRoy2226/Raman-Greens.git
```

### 2️⃣ Navigate to project folder

```bash
cd Raman-Greens
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the development server

```bash
npm run dev
```
